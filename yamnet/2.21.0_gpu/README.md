# yamnet

[yamnet](https://www.kaggle.com/models/google/yamnet/tensorFlow2/yamnet/1), 
uses Tensorflow 2.21.0 for GPU.


## Quick start

### Directories

Create the following directories where you want to start the Docker image from:

```
mkdir cache
mkdir config
```

### Inhouse registry

* Pull and run image (adjust volume mappings `-v`):

  ```bash
  docker run --rm --gpus=all --shm-size 8G --net=host \
    -v `pwd`:/workspace \
    -v `pwd`/cache:/cache \
    -v `pwd`/config:/config \
    -it harbor.cms.waikato.ac.nz/public/tensorflow/tf-yamnet:2.21.0_gpu
  ```

### Docker hub

* Pull and run image (adjust volume mappings `-v`):

  ```bash
  docker run --rm --gpus=all --shm-size 8G --net=host \
    -v `pwd`:/workspace \
    -v `pwd`/cache:/cache \
    -v `pwd`/config:/config \
    -it waikatodatamining/tf-yamnet:2.21.0_gpu
  ```

### Build local image

* Build the image from Docker file (from within /path_to/yamnet/2.21.0_gpu)

  ```bash
  docker build -t yamnet .
  ```
  
* Run the container

  ```bash
  docker run --rm --gpus=all --shm-size 8G --net=host \
    -v `pwd`:/workspace \
    -v `pwd`/cache:/cache \
    -v `pwd`/config:/config \
    -it yamnet
  ```
  `/local/dir:/container/dir` maps a local disk directory into a directory inside the container


## Publish images

### Build

```bash
docker build -t yamnet:2.21.0_gpu .
```

### Inhouse registry  

* Tag

  ```bash
  docker tag \
    yamnet:2.21.0_gpu \
    harbor.cms.waikato.ac.nz/public/tensorflow/tf-yamnet:2.21.0_gpu
  ```
  
* Push

  ```bash
  docker push harbor.cms.waikato.ac.nz/public/tensorflow/tf-yamnet:2.21.0_gpu
  ```
  If error "no basic auth credentials" occurs, then run (enter username/password when prompted):
  
  ```bash
  docker login harbor.cms.waikato.ac.nz
  ```

### Docker hub  

* Tag

  ```bash
  docker tag \
    yamnet:2.21.0_gpu \
    waikatodatamining/tf-yamnet:2.21.0_gpu
  ```
  
* Push

  ```bash
  docker push waikatodatamining/tf-yamnet:2.21.0_gpu
  ```
  If error "no basic auth credentials" occurs, then run (enter username/password when prompted):
  
  ```bash
  docker login
  ``` 


### Requirements

```bash
docker run --rm --pull=always \
  -it harbor.cms.waikato.ac.nz/public/tensorflow/tf-yamnet:2.21.0_gpu \
  pip freeze > requirements.txt
```


## Permissions

When running the docker container as regular use, you will want to set the correct
user and group on the files generated by the container (aka the user:group launching
the container):

```bash
docker run -u $(id -u):$(id -g) -e USER=$USER ...
```
